# Work It App

App to track workouts and progress.

## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Open the Expo Go app and scan the QR code.

## Development

If you make any changes to the database schema, run the following script to update Drizzle migrations:

```bash
  npm run drizzle:generate
```
